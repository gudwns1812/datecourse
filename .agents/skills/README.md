# Project Skills

This project now includes the following local Codex skills under `.agents/skills`.

## Installed

- Existing project skill: `frontend-coder`
- Added system skills: `imagegen`, `openai-docs`, `plugin-creator`, `skill-creator`, `skill-installer`
- Added workflow skills: `figma-generate-design`, `figma-implement-design`, `figma-use`, `frontend-skill`, `playwright`, `screenshot`, `vercel-deploy`

## Recommended For This Project

- `frontend-coder`: general frontend implementation in the Next.js app
- `frontend-skill`: stronger landing page and UI art direction
- `playwright`: browser automation for UI checks and flow debugging
- `screenshot`: desktop-level capture when browser or tool-native capture is not enough
- `vercel-deploy`: preview deployments for frontend review
- `figma-implement-design`: implementing Figma designs into code
- `figma-use` + `figma-generate-design`: pushing screens or layouts into Figma using design-system-aware workflows
- `openai-docs`: OpenAI API or model-selection questions using official docs
